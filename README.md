# Blockchain Voting System

## Prerequisites
- Node.js v16+ and npm
- MetaMask (or other Web3) browser extension. Don't need to download it right now, can do later.

## Installation & Setup

1. **Clone the repo**

2. **Server**
   ```bash
   cd server
   npm install
   npm start    # runs on port 3001 by default
   ```

3. **Client**
   ```bash
   cd client
   npm install
   npm run dev  # runs on http://localhost:3000
   ```
